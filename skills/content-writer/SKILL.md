## Universal Rules (both businesses)

- First line must be a hook — no fluff openers ("In today's world…", "We all know…", "Have you ever…")
- End with ONE clear CTA only
- Stay inside character limits per channel
- Use specifics over generics — name a centre / location / class / time
- Use active voice
- Apply ≥2 psychology triggers from the resolved psychology-triggers file
- Australian spelling (centre, programme, colour)

## Content Formulas

### Instagram Reel Script (15-30 sec)

```
Line 1 (Hook):       Bold statement or question — max 8 words
Line 2 (Problem):    Pain point the viewer feels
Line 3 (Solution):   How [active business] solves it (specific)
Line 4 (Proof):      Specific detail (price / facility / member or family count)
CTA:                 One action, direct
```

### Instagram Caption

```
Hook (1 line) → Context (2-3 sentences) → CTA (1 line)
Hashtags: 5-8 max, relevant only
```

For MWCC: avoid generic parenting hashtags (`#parenting`, `#perthmum`) that pull low-intent audience. Use `#earlyLearning`, `#PerthChildcare`, `#OSHC[suburb]`, `#CCSapproved`.

### Facebook Post

```
Conversational opener → Story or benefit → Community angle → CTA
Longer form OK — local audience responds to specific suburb / school references.
```

### Ad Copy (Meta)

**CB247:**
```
Headline:    Max 40 chars — benefit-led
Description: Max 125 chars — price anchor + differentiator
CTA Button:  "Learn More" or "Sign Up"
```

**MWCC:**
```
Headline:    Max 40 chars — trust-led (e.g. "Qualified care in [suburb]")
Description: Max 125 chars — CCS reminder + concrete next step
CTA Button:  "Book Tour" / "Learn More" (avoid "Sign Up" — too commercial for childcare)
```

### Email Subject Line

- Max 40 chars (Gmail truncates beyond)
- Avoid "FREE", "GUARANTEE", "!!" — spam triggers + MWCC compliance flag
- Question or specific number performs best
- Test 2 variants per send when possible

### SMS (CB247 only — MWCC doesn't use SMS for marketing)

- Max 160 chars
- Front-load the offer
- End with shortlink

## Competitor Positioning in Copy

### CB247
- vs Revo: "Same price, but we have [specific facility]"
- vs Ryderwear: "Train hard AND [family/variety benefit]"
- vs Anytime: "Local gym, premium facilities, half the price"
- General: Never name competitors directly in public content

### MWCC
- vs Goodstart / Nido (national chains): position on local ownership + educator continuity ("our educators have been with us 3+ years")
- vs aggregators (KindiCare, Care for Kids): position on direct relationship ("talk to the centre director, not a search aggregator")
- vs council OSHC: position on continuity ("LDC + OSHC under one group at Midvale + Seville Grove")
- General: NEVER name competitors directly. Disparaging language ("better than [X]") auto-blocks at compliance gate.
