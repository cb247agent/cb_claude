## What to Extract Per Ad

| Data Point | Why It Matters |
|------------|----------------|
| Ad creative | Visual hook, format, branding |
| Ad copy | Headlines, body text, CTAs |
| Offer structure | What they're promoting |
| Targeting stated | Who the ad is reaching |
| Run duration | How long the campaign ran |
| Estimated spend | Approximated from library data |
| Landing page | Where the ad leads |

## Ad Classification Framework

| Type | Description | Example |
|------|-------------|---------|
| Prospecting | Cold audience, new member acquisition | "Join from $X/week" |
| Retargeting | Warm audience, past visitors | "Welcome back offer" |
| Brand Awareness | Top-of-funnel, reach | "We're celebrating 10 years" |
| Promotional | Price-focused | "50% off first month" |
| Seasonal | Event/timing-based | "New Year, new you" |
