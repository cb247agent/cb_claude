## CB247's 5 Core ICPs

### ICP 1: FIFO Worker
**The "Train on Your Swing" Segment**

| Attribute | Detail |
|-----------|--------|
| Age | 28-50 |
| Gender | All (male skew 65/35) |
| Location | Malaga, Ellenbrook, NOR Perth (10km radius) |
| Occupation | Mining, construction, oil & gas |
| Schedule | Rotating rosters (7/7, 8/6, 14/14) |
| Income | $80K-$150K+ (high disposable) |
| Pain Points | Odd hours, contracts don't fit roster, gym closes when they're off |
| Key Message | "Train when you're home. Freeze when you're gone." |
| Facilities That Matter | 24/7 access, FIFO freeze, recovery (sauna/ice bath between shifts) |
| Meta Targeting | FIFO, mining, fly-in fly-out, fitness, gym, Perth, Western Australia |
| Content Angle | Schedule flexibility, no lock-in, recovery after swing |
| Objections | "My schedule doesn't fit gym memberships", "I'll be away for weeks" |
| Price Sensitivity | Medium — can afford premium, but want value |
