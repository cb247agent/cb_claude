## Implementation Instructions

**Webflow:**  
Open each page → Add a Text Block at the top of main content area → Set heading to H1 → Type the recommended H1 text → Save + Publish.

**WordPress:**  
Edit each page → Ensure the Page Title field is set (this auto-generates H1) OR add a Heading block at top of content, set to H1.

**Important Rules:**
- Only ONE H1 per page — do not add multiple H1s
- H1 should appear above the fold (visible without scrolling)
- H1 can be different from the SEO Title (Title Tag) — they serve different purposes
- H1 = what the visitor sees on the page
- Title Tag = what appears in Google search results
